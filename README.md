# HTML-CSS-Project
This project is a fully responsive landing page built using **HTML** and **CSS**.  
It is a recreation of the *Launch* template from Colorlib and was created as part of my frontend development learning journey.

## Features
- Responsive navigation bar  
- Hero section with text & illustration  
- Features and services sections  
- Pricing plans layout  
- Testimonials section  
- About page section  
- Blog/news section  
- Contact form layout  
- Fully structured with clean and reusable CSS

## Purpose of This Project
I built this project to:
- Strengthen my HTML & CSS skills  
- Understand how to structure a complete webpage  
- Practice layout techniques (flexbox, spacing, alignment)  
- Learn how to improve UI using icons, typography, and colors  

## Technologies Used
- **HTML5**
- **CSS3**
- **Font Awesome Icons**
- **Bootstrap Icons**
